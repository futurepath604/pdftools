Files will be assigned sequential IDs (e.g., File 1, File 2)

Format: FileID[PageRange] separated by commas.
Example: 1[1-3], 2[5,6], 1[4] means take Pages 1 to 3 from File 1, then Pages 5 & 6 from File 2, and finally Page 4 from File 1.
